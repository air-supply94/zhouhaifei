## 下载

```bash
yarn global add yo@3.1.0
# 下载最新generator-zhouhaifei-react
yarn global add generator-zhouhaifei-react
# 存在则更新
yarn global upgrade generator-zhouhaifei-react --latest
```

## 生成

```bash
# mac 、linux
mkdir 目录名称 && cd 目录名称
# window则自行创建目录,然后进入目录
yo zhouhaifei-react
```
